## 中文

这是 0.2 大版本的首次公开发布，累积了不少更新。

### 新功能

- **菜单栏可以替代系统音量图标**：状态颜色一眼看清在播 / 静音 / 待机，每次调音都有 OSD 显示。首启会引导你把系统的音量图标隐藏掉。
- **预设（Pro）**：把常用的设备组合保存成预设，一键切换。比如「客厅 + 蓝牙耳机」「电影模式」「会议模式」。
- **键盘音量键和滚轮调音量（Pro）**：F11 / F12 / 静音键直接控制多设备同播的总音量；菜单栏图标和 popover 上滚动鼠标或触控板都能调；Shift + Option 微调步长跟系统一致。
- **看到当前 AirPlay 设备**：如果你已经用系统菜单切到了 AirPlay 接收器，它会出现在 Tutti 的设备列表里，像普通设备一样能调音量。
- **自动更新**：以后有新版本会自动提示你升级，不用再来 GitHub 翻。

### 修复

- 切换主题时 Settings 窗口偶尔会卡死在原主题。
- 退出 Tutti 时会自动清理为多设备同播创建的虚拟设备。

### Settings 改版

设置窗口三个标签页（通用 / 许可 / 关于）风格统一，9 种语言（简繁中文、英、日、韩、法、德、意、西）全部覆盖。

## English

First public release of the 0.2.x series — accumulated updates from three versions.

### New

- **Menu bar replacement for macOS's audio icon** — three-state color tells you what's playing, what's muted, and what's idle at a glance. OSD pops on every volume change. First launch walks you through hiding the system icon.
- **Profiles (Pro)** — save device combinations as named profiles and switch with one click. Think "Living room + AirPods", "Movie", "Meeting".
- **Volume keys and scroll wheel (Pro)** — F11 / F12 / Mute drive the multi-device output. Scroll on the menu bar icon or popover does the same. Shift+Option fine-grain steps match the system.
- **Active AirPlay surfacing** — once macOS has routed audio to an AirPlay receiver, it shows up in Tutti's device list, ready to be selected like any other device.
- **Auto-update** — new versions notify you automatically. No more checking GitHub.

### Fixes

- Settings window occasionally stuck on the previous theme when switching.
- Tutti now cleans up its aggregate device on quit.
